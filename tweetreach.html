TweetReach

TweetReach is a twitter analytic app developed by the California based Union Metrics. It was launched in 2009 as a snapshot report app and has evolved into a full featured twitter analytics app. It provides both real time visualisation and data extraction. It is focused on the social media marketing and branding crowd. Perhaps because of its popularity it has very expensive plans. A feature rich app if you can afford it and have tablets to use it.

The Good:

  • Single page UI
  • Extremely simple to use yet offer complex functionality
  • Data generated belong to the user
  • No third party involvement

The (not so) Bad:

  • Customer support is differentiated through payment plans
  • No refunds for partial months of use

The Ugly:

  • Free version has reduced functionality
  • Very expensive compared to competition
  • Optimised only for tablet use
